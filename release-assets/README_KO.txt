나의 여름방학 포터블 한글패치 image-kr v0.1.3

적용 방법

1. 원본 ISO를 이 폴더에 복사합니다.
2. 파일 이름을 `Boku no Natsuyasumi Portable.iso`로 맞춥니다.
3. `apply_iso_patch.bat`를 실행합니다.
4. `Boku no Natsuyasumi Portable KR image-kr v0.1.3.iso`가 생성됩니다.

주의

- 원본 ISO MD5는 `B4D363D59CB87E25AB76AFC5384CCA31`이어야 합니다.
- 이 버전은 pause-guard v0.1.2의 대사 안정화 위에 타이틀, 시작 메뉴, 저장 데이터 이미지, 주요 게임 메뉴 이미지를 한글화한 빌드입니다.
- ISO 전체를 배포하지 않고 xdelta 패치만 배포합니다.
